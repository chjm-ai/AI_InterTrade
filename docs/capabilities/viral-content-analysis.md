# 爆款内容拆解

## 结论状态

`待验证`

## 目标

抓取竞品或行业爆款内容，输出适合外贸老板理解的拆解结果，而不是单纯原始数据。

## 候选技术方案

### 方案一：Apify 平台（推荐）

**优点：**
- 现成 Actor 生态（TikTok、Instagram、YouTube Shorts、LinkedIn 等）
- 支持代理池和反爬绕过
- 数据格式标准化（JSON/CSV/API）
- 可视化调试和监控
- Serverless，无需自建服务器

**成本估算：**
- 基础版：$49/月（含 $49 计算积分）
- TikTok scraper：约 $0.5-2/1000 视频
- 按量付费，无固定服务器成本

**技术验证点：**
- TikTok / Instagram 数据抓取字段完整性
- 评论区数据深度（嵌套评论）
- 数据更新频率限制

### 方案二：自建爬虫 + LLM 分析

**技术栈：**
- 爬虫：Playwright / Scrapy + 代理池
- 存储：Supabase / PostgreSQL
- 分析：OpenAI API / Claude API
- 部署：Fly.io / Railway

**优缺点：**
- 优点：完全可控、无平台依赖
- 缺点：开发周期长、维护成本高、反爬对抗难度大

**成本估算：**
- 代理费用：$50-200/月
- 服务器：$20-50/月
- LLM API：$20-50/月（按量）
- 合计：$90-300/月 + 开发维护人力

## 平台选择建议（外贸场景）

优先级排序：

1. **TikTok** - 全球流量最大，适合 B2C 和消费品外贸
2. **Instagram Reels** - 欧美主流，适合时尚、家居类目
3. **LinkedIn** - B2B 外贸核心渠道
4. **YouTube Shorts** - 长视频短内容兼顾

## 要重点验证的点

- [ ] 抓取价格（Apify vs 自建成本对比）
- [ ] 抓取成功率与稳定性（不同平台差异）
- [ ] 返回数据格式是否适合做结构化分析
- [ ] 关键字段完整性（点赞、评论、分享、标签、音乐）
- [ ] 评论数据的层级和情感分析可行性
- [ ] 是否能沉淀为网页版能力或 skill

## 核心风险与应对

### 风险1：爬虫稳定性受平台影响大
**应对：**
- Apify 提供自动代理轮换和反爬策略
- 设置监控告警，失败自动重试
- 备选数据源（多平台冗余）

### 风险2：不同平台字段不统一
**标准化方案：**

```
统一数据模型：
- content_id: 内容唯一ID
- platform: 平台标识
- author: 作者信息
- publish_time: 发布时间
- content_text: 文案内容
- media_urls: 媒体文件链接
- metrics: {
  - views: 播放量
  - likes: 点赞数
  - comments: 评论数
  - shares: 分享数
  - engagement_rate: 互动率
}
- tags: 标签数组
- music_info: 音乐信息（如有）
```

### 风险3：内容分析深度有限
**应对：**
- 结合 LLM 进行多维度分析（钩子、结构、CTA）
- 视频内容增加语音转文本分析
- 竞品对比分析而非单点分析

## 建议验证路径

### 阶段1：技术可行性验证（1-2天）
1. 注册 Apify 账号，获取免费积分
2. 选择 TikTok Scraper Actor 进行测试
3. 抓取 50-100 条目标类目爆款视频
4. 检查返回字段完整性和数据质量

### 阶段2：分析模型验证（2-3天）
1. 设计分析维度（文案结构、钩子、CTA、互动点）
2. 使用 LLM API 进行样本分析
3. 提炼可复用的分析模板
4. 输出分析报告样例

### 阶段3：产品化验证（3-5天）
1. 搭建简单的分析报告生成页面
2. 邀请 2-3 位外贸老板试用并反馈
3. 根据反馈调整分析维度和展示方式
4. 评估是否值得投入完整开发

### 验证检查清单

- [ ] Apify 账号注册成功

## 分析报告结构示例

### 单视频拆解报告

```
📊 基础数据
- 播放量：X万 | 点赞：X万 | 评论：X千 | 分享：X千
- 互动率：X%（行业平均 X%）
- 发布时间：X月X日 X时（黄金时段）

🎬 视频结构拆解
- 前3秒钩子：[描述具体手法]
- 核心卖点：[提炼的卖点]
- 信任建立：[如何建立信任]
- CTA动作：[引导的下一步]

✍️ 文案亮点
- 标题结构：[分析标题模式]
- 文案节奏：[短句/长句/emoji使用]
- 关键词：[高频词提取]

💡 可复制元素
- 钩子模板：[提供3个类似钩子]
- 拍摄手法：[可模仿的拍摄技巧]
- 音乐选择：[音乐风格/情绪]

🎯 适配建议
- 适配产品：[你的哪些产品适合用]
- 调整方向：[需要修改的部分]
- 预期效果：[合理预期]
```

### 竞品批量对比报告

```
📈 数据概览
- 分析样本：X条视频
- 时间跨度：X月-X月
- 平均互动率：X%
- 爆款标准：播放量>X万

🏆 爆款特征总结
1. 最佳发布时段：[数据支撑]
2. 平均视频时长：[X秒]
3. 高频标签TOP10：[列表]
4. 常用BGM类型：[分类统计]

🎪 内容模式分类
- 痛点型：X% | 案例：[代表视频]
- 对比型：X% | 案例：[代表视频]
- 教程型：X% | 案例：[代表视频]
- 展示型：X% | 案例：[代表视频]

💰 转化分析（如可获取）
- 评论区热词：[提取购买意向词]
- 用户画像：[从评论分析]
- 竞争程度：[同类内容密度]

📋 行动建议
- 立即执行：[低成本高回报的动作]
- 短期规划：[1个月内可落地]
- 长期布局：[需要准备的方向]
```

## 最终交付形态建议

优先级从高到低：

1. **分析报告** - 最易验证价值，最低成本
2. **网页版分析面板** - 可交互、可持续使用
3. **Skill 形式** - 嵌入工作流，自动化执行
